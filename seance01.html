Séance 1 - Rappel

Objectifs

  1. Importer des fonctions à partir de modules
  2. Appeler des fonctions et leur passer des arguments
  3. Saisir des données au clavier
  4. Utiliser une structure conditionnelle
  5. Utiliser une structure itérative complète

Activités

Soit le programme Python suivant:

from turtle import *
left(25)
forward(100)
mainloop()

Explication

Travail à faire

  1. Utiliser les fonctions précédentes pour dessiner un carré.
  2. Modifier le programme pour dessiner un rectangle.
  3. Dessiner le rectangle avec une longueur et une largeur données.

    Code snippet

    L'instruction suivante permet de saisir une donnée au clavier :

    long = int(input('Longueur : '))

    Explication

  4. Dessiner un polygone régulier avec un nombre de cotés donné.
    • Si nc = 3 ⇒ Triangle équilatéral
    • Si nc = 4 ⇒ Carré
    • Si nc = 5 ⇒ Pentagone régulier
    • Si nc = 6 ⇒ Hexagone régulier

    Code snippet

    for _ in range(nc):
    forward(long)
    left(360 / nc)

    Explication
