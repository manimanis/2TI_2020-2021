Chaines (résolu)

Ecrire l'algorithme d'un programme saisit le nom et le prénom d'un élève dans une variable nom, le nom doit être une chaine non vide qui contient au moins un espace, puis affiche :

  • Le nombre de caractères qui constituent la chaine nom
  • Le premier, la dernier et le caractère au milieu de nom
  • Les initiales de nom
  • Extrait le nom et le prénom de la variable nom
Donner votre nom ? Mahmoud Letaief
Ce nom contient 15 caractères
Le premier caractère est : 'M'
Le dernier caractère est : 'f'
Le caractère au milieu est : ' '
Les initiales sont : ML
Le prénom est : Mahmoud
Le nom est : Letaief

Correction

Algorithme Chaines
Début
  Répéter
    Ecrire('Donner votre nom ? ')
    Lire(nom)
  Jusqu'à (nom ≠ "") et (pos(" ", nom) ≠ -1)

  l ← long(nom)
  Ecrire("Ce nom contient", l, "caractères")

  Ecrire("Le premier caractère est : '" + nom[0] + "'")
  Ecrire("Le dernier caractère est : '" + nom[l - 1] + "'")
  Ecrire("Le caractère au milieu est : '" + nom[l div 2] + "'")

  pe ← Pos(" ", nom)
  
  init ← nom[0] + nom[pe + 1]
  Ecrire("Les initiales sont :", init)

  prenom ← sous_chaine(nom, 0, pe)
  nom ← efface(nom, 0, pe+1)
  Ecrire("Votre nom est :", nom)
  Ecrire("Votre prénom est :", prenom)
Fin

Palindrôme

Ecrire l'algorithme d'un programme qui permet de vérifier si une chaine ch, non vide, est palindrome.

Une chaine palindrome est une chaine qui reste la même lorsqu'elle est inversée, exemples : RADAR, AZIZA, ALA, KAYAK, ETE, TOT, etc.

Donner une chaine ? RADAR
RADAR est palindrome

Github Pages

Github est un service web d'hébergement et de gestion de développement de logiciels utilisant le logiciel de gestion de versions Git.

Lorsqu'on héberge un projet sur Github son code source sera accessible à travers l'adresse URL :

https://github.com/[utilisateur]/[projet]/

Exemple : https://github.com/manimanis/4ECO/, ici manianis est le nom d'utilisateur, et 4ECO est le nom du projet

Les projets web peuvent bénéficier de l'hébergement gratuit à travers le service Github pages. Ainsi, un projet sera accessible à l'adresse URL :

https://[utilisateur].github.io/[projet]/

Exemple : https://manimanis.github.io/4ECO/

Travail Demandé

On demande d'écrire l'algorithme d'un programme qui :

  • Saisit une adresse URL Github ayant l'une des deux formes précédentes :

    • https://github.com/[utilisateur]/[projet]/
    • https://[utilisateur].github.io/[projet]/
  • Convertit l'adresse saisie dans l'autre seconde forme :
    • https://github.com/[utilisateur]/[projet]/ ⇆ https://[utilisateur].github.io/[projet]/
    • Et inversement

Github repo : https://github.com/manimanis/4ECO/
Adresse Github ? https://github.com/manimanis/4ECO/

On souhaite écrire un algorithme qui permet de raccourcir ces adresses. Ainsi, l'adresse précédente devient :
